Program to find the maximum and minimum element in an array

Using CPP language

CPP Code


                #include 'iostream'
                using namespace std;

                int main()
                {
                        int n;
                        cout<<"Enter Size of Array : ";
                        cin>>n;
                        int ar[n];
                        cout<<" Enter Array Elements : ";
                        for(int i=0 ; i>ar[i];
                        }
                        int min = ar[0] , max =ar[0];
                        for(int i=0 ; i< n ; i++ )
                        {
                            if(ar[i] > max)
                            {
                                max = ar[i];
                            }
                            if(ar[i] < min)
                            {
                                min = ar[i];
                            }
                        }
                        cout<<" Max : " << max ;
                        cout<< " Min : " << min ;
                }

Sample Input

                    Enter Size of Array : 4
                    Enter Array Elements : 2 3 4 1

Sample Output


                Max : 4
                Min : 1

Variables Used in Program

                    int n      number of elements in array
                    int ar[]   array 
                    int max    maximum element
                    int min    minimum element

Time Complexity

                Complexity : O(n) 
                Above Solution Provides Answere in Linear Time Complexity
                

                Another Approach is by sorting technique
                Which will provide solution in O(n2) time
